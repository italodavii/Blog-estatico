Desenvolvimento WEB

Voltar

Este trabalho de Desenvolvimento Web tem como objetivo aplicar os conhecimentos adquiridos em HTML e CSS, criando páginas web estruturadas e estilizadas.


Durante o projeto, foram explorados conceitos de marcação semântica, organização de conteúdo, e design responsivo, visando a criação de interfaces atraentes e funcionais para diferentes dispositivos.


O foco foi na compreensão da estrutura básica de uma página web e na utilização de CSS para controlar o layout, cores, tipografia e outros aspectos visuais.

Página feita por Italo David, Marcus Vinicius e Erick Santos
alunos de Eng. de Software e Sistemas de Informação.
